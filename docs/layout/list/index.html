List

List is a flex solution to space it's childrens in a column/row layout. Can also be used to automatically style lists of buttons.

Options

Custom

In theming code use .list_btn--all({}); to style all .list > li > .btn or .list_btn({}); to style all .list:not(.custom) > li > .btn.

.list_btn({}); .list_btn--all({});

Space

Lists automatically add horizontal and vertical spacing to the contents. You can specify a different one with this classes/mixins.

Syntax Example
Class .list-space--{size} .list-space--small
Class responsive .list-space--{size}-{breakpoint} .list-space--small-sm
Mixin .list-space({size}) .list-space(small)
Mixin responsive .list-space({size}, {breakpoint}) .list-space(small, sm)
You can set list's spaces in the _variables.less file. Read the documentation of list's variables.
.list-space--none
.list-space--tiny

Alignment

Lists automatically has left aligment. You can specify a different one with this classes/mixins.

You can also set others flex alignments like .list_btn({ > span { justify-content: space-between; } });.

Syntax Example
Class .list--left .list--left
Class responsive .list--left-{breakpoint} .list--left-sm
Mixin .list--left() .list--left()
Mixin responsive .list--left({breakpoint}) .list--left(sm)
.list--left
.list--center
.list--right
