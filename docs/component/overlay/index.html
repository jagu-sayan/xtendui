Usage

Use this markup to create a overlay. [data-xt-overlay] must be initialized by javascript.
For customizing the interactions read the documentation of toggle overlay.
Overlays are moved to body to prevent z-index problems.

Variables

Read the documentation of component's variables.

Options

Variants

Optional classes for assigning variant.

You can change the default variant inside the .overlay selector in /theme/_component.less.

.overlay-primary .overlay-secondary .overlay-white

Sizes

Optional classes for assigning width.

You can change the default size inside the .overlay selector in /theme/_component.less.

Syntax Example
Class .overlay-size-{size} .overlay-size-small
Mixin .overlay-size({size}) .overlay-size(small)
You can set overlay's size in the _variables.less file. Read the documentation of overlay's variables.
.overlay-size-tiny .overlay-size-small .overlay-size-medium .overlay-size-big .overlay-size-giant .overlay-size-full
