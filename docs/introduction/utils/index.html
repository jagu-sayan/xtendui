Flexbox

Flexbox utils are used to assign flexbox options to flex and list.

Direction

Set children's direction inside the flexbox.

.flex-direction--row .flex-direction--row-reverse .flex-direction--column .flex-direction--column-reverse

Syntax Example
Class .flex-direction--{type} .flex-direction--row
Class responsive .flex-direction--{type}-{breakpoint} .flex-direction--row-sm
Mixin .flex-direction({type}) .flex-direction(row)
Mixin responsive .flex-direction({type}, {breakpoint}) .flex-direction(row, sm)

Wrap

Set children's wrapping inside the flexbox.

.flex-wrap--wrap .flex-wrap--wrap-reverse .flex-wrap--nowrap

Syntax Example
Class .flex-wrap--{val} .flex-wrap--wrap
Class responsive .flex-wrap--{val}-{breakpoint} .flex-wrap--wrap-sm
Mixin .flex-wrap({val}) .flex-wrap(wrap)
Mixin responsive .flex-wrap({val}, {breakpoint}) .flex-wrap(wrap, sm)

Justify content

Set children's horizontal alignment inside the flexbox.

.justify-content--flex-start .justify-content--flex-end .justify-content--center .justify-content--space-between .justify-content--space-around

Syntax Example
Class .justify-content--{type} .justify-content--flex-start
Class responsive .justify-content--{type}-{breakpoint} .justify-content--flex-start-sm
Mixin .justify-content({type}) .justify-content(start)
Mixin responsive .justify-content({type}, {breakpoint}) .justify-content(start, sm)

.align-items--flex-start .align-items--flex-end .align-items--center .align-items--baseline .align-items--stretch

Align self

Set item's vertical alignment inside the flexbox. If set to auto computes to the parent's align-items value.

.align-self--flex-start .align-self--flex-end .align-self--center .align-self--baseline .align-self--stretch .align-self--auto

Syntax Example
Class .align-self--{type} .align-self--flex-start

Mixin .align-self({type}) .align-self(start)

Order

Set item's order inside the flexbox.

Syntax Example
Class .order-{number} .order-1
Class responsive .order-{number}-{breakpoint} .order-1-sm
Mixin .order({number}) .order(1)
Mixin responsive .order({number}, {breakpoint}) .order(1, sm)
You can set orders in the _variables.less file. Read the documentation of flex's variables.
