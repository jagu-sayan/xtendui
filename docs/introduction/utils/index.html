Flexbox

Flexbox utils are used to assign flexbox options to flex and list.

Direction

Set children's wrapping inside the flexbox.

.flex-row .flex-row-reverse .flex-column .flex-column-reverse

Syntax Example
Class .flex-{type} .flex-row
Class responsive .flex-{type}-{breakpoint} .flex-row-sm
Mixin .flex-{type}() .flex-row()
Mixin responsive .flex-{type}({breakpoint}) .flex-row(sm)

Wrap

Set children's wrapping inside the flexbox.

.flex-wrap .flex-wrap-reverse .flex-nowrap

Syntax Example
Class .flex-wrap .flex-wrap
Class responsive .flex-wrap-{breakpoint} .flex-wrap-sm
Mixin .flex-wrap() .flex-wrap()
Mixin responsive .flex-wrap({breakpoint}) .flex-wrap(sm)

Justify content

Set children's horizontal alignment inside the flexbox.

.justify-content-start .justify-content-end .justify-content-center .justify-content-between .justify-content-around

Syntax Example
Class .justify-content-{type} .justify-content-start
Class responsive .justify-content-{type}-{breakpoint} .justify-content-start-sm
Mixin .justify-content-{type}() .justify-content-start()
Mixin responsive .justify-content-{type}({breakpoint}) .justify-content-start(sm)

Set children's vertical alignment inside the flexbox.

.align-items-start .align-items-end .align-items-center .align-items-baseline .align-items-stretch

Syntax Example
Class .align-items-{type} .align-items-start

Align self

Set item's vertical alignment inside the flexbox. If set to auto computes to the parent's align-items value.

.align-self-start .align-self-end .align-self-center .align-self-baseline .align-self-stretch .align-self-auto

Syntax Example
Class .align-self-{type} .align-self-start
Class responsive .align-self-{type}-{breakpoint} .align-self-start-sm
Mixin .align-self-{type}() .align-self-start()
Mixin responsive .align-self-{type}({breakpoint}) .align-self-start(sm)

Order

Set item's order inside the flexbox.

Syntax Example
Class .order-{number} .order-1
Class responsive .order-{number}-{breakpoint} .order-1-sm
Mixin .order({number}) .order(1)
Mixin responsive .order({number}, {breakpoint}) .order(1, sm)
You can set orders in the _variables.less file. Read the documentation of flex's variables.
