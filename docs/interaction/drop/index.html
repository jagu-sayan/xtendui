Usage

Use this markup to create a drop. [data-xt-drop] must be initialized by javascript.
Instead of using html arguments you can initialize drop by javascript using the same options syntax.
For customizing the interactions read the documentation of toggle drop.

Defaults

Here are the defaults options for drop.

  
XtDrop.defaults = {
  "targets": ":scope > .drop",
  "controls": ":scope > a, :scope > button",
  "class": "active",
  "instant": {"elements": true},
  "on": "click",
  "toggle": true,
  "min": 0,
  "max": 1,
  "closeOutside": "body",
  "aria": true
};

Drop of lists

The natural way to use drops is to use them with lists of buttons.

Read the documentation of list of buttons.

Buttons styles

.list-drop > li > a and .list-drop > li > button are styled automatically, If you want to customize the buttons use the .btn class instead.

Read the documentation of list's buttons styles.
In theming code use .list-btn-default({}); to style a and button. To style also if they have the .btn class use .list-btn-all({});.

Options

Variants

Optional classes for assigning variant. .list-drop has drop's list default options.

You can change the default variant inside the .drop selector in /theme/_component.less.

.drop-primary .drop-secondary .drop-white

Position

Optional classes for assigning position.

You can change the default position inside the .drop selector in /theme/_component.less.

.drop-bottom .drop-left .drop-right .drop-center
.drop-top .drop-left .drop-right .drop-center
.drop-left .drop-right .drop-middle .drop-center
.drop-before .drop-top .drop-bottom .drop-middle
.drop-after .drop-top .drop-bottom .drop-middle
.drop-middle-top .drop-middle-bottom

Special

Special optional modifiers classes.

.drop-nodesign .drop-squared .drop-noborder

Addons

Static

Drop position relative to closest element with position: relative;.

For limiting event inside a selector use .event-limit.

.drop-outer-static
.drop-outer-static .drop-top
.drop-outer-static .middle

Backdrop

Option to have a backdrop.

data-xt-drop='{"backdrop": true}' .over-backdrop

Event

You can specify on and off events for the drop. Default is "on": "click".

"on": "mouseenter" "off": "mouseleave"

Dimensions

Drop with max sizes with automatic scrollbar on overflow.

.drop-max-height
.drop-max-width
.drop-max-height .drop-max-width

Nested and Disable

For limiting event inside a selector use .event-limit
To disable drop use .drop-disable useful within media queries
