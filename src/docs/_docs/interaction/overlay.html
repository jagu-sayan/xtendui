Usage

Use this markup to create a overlay.

You can use this markup to create a overlay with no toggle. Be sure to add .overlay_content.event-limit.

You can use this markup to create a overlay with no id.

You can initialize overlay by javascript omitting [data-xt-overlay].

  
    let self = Xt.init('xt-overlay', document.querySelector('.my-overlay'), {
      // option
    });
    self.unmount = function() {
      self = null;
    };
Overlays are moved to body to prevent z-index problems.

Options

Variants

Optional classes for assigning variant.

You can change the default variant inside the .overlay selector in /theme/_component.less.

.overlay--primary .overlay--secondary .overlay--white

Sizes

Optional classes for assigning width.

You can change the default size inside the .overlay selector in /theme/_component.less.

Syntax Example
Class .overlay--{size} .overlay--small
Mixin .overlay--size({size}) .overlay--size(small)
You can set overlay's size in the _variables.less file. Read the documentation of overlay's variables.
.overlay--tiny .overlay--small .overlay--medium .overlay--big .overlay--giant .overlay--full
