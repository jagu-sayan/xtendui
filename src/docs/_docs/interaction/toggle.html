Usage

Use this markup to create a toggle.

You can initialize toggle by javascript omitting [data-xt-toggle].

  
    let self = Xt.init('xt-toggle', document.querySelector('.my-toggle'), {
      // option
    });
    self.unmount = function() {
      self = null;
    };

Mode

Preferred mode is multiple, but unique is useful when triggering targets outside the scope, like overlay where the targets are moved to body.

For toggle utils classes read the documentation of toggle utils.

Multiple

You can create a "mutiple" mode by assigning [data-xt-toggle] to a scope element, elements and targets must be inside the scope. Check toggle's defaults for the default elements and targets.

With multiple mode you can't specify targets with #id.
With multiple targets, the ones activated is the one with the same index of the toggled element. Nested targets are filtered out

Class

Toggled class is by default .active.

You can specify classes to toggle with class
Target

You can start with toggled elements and targets just by adding all of the classes to the elements or targets you want activated. Following this activation the min option will take effect.

If you want to restart the component with the initial starting element use self.initStart();

Min and max

Inside a manager you can specify min or max activations. Default is "min": 0 and "max": 1.

Use "max": "Infinity" for no limits.

The required min activation is fulfilled on initialization

Jump

Use "jump": true to enable clicking on a target to jump to them.

Navigation

You can add navigation with the navigation option, set the amount to add to the current activation with data-xt-nav="value"

Use "loop": false to disable loop.

Timing

Activation/deactivation automatically waits for the duration of animations and transitions on .active and .out
To set duration (if you are animating by js for example) use "durationOn": Milliseconds "durationOff": Milliseconds
To add a delay before duration use "delayOn": Milliseconds "delayOff": Milliseconds
You can use a function for "delayOn" and "delayOff" like this: function(current, total) {return current * 75}. You can also use a string "return current * 75".
To disable queue and automatic duration use "instant": true

Javascript

You can animate with javascript, just use Xt events and remember to set "durationOn": Milliseconds and "durationOff": Milliseconds

Other

Use "keyboard": {"selector": "object"} or "keyboard": {"selector": Query} to enable keyboard navigation on focus
