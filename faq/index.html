Browsers bugs

Fixed position

On all browsers, when using transforms on elements or within a running animation, all childs with position: fixed; behave likes position: absolute;.

Transitions and animations

On safari, when using both transitions and animations on elements, the animations runs incorrectly sometimes, use just transitions or animations don't mix them.
