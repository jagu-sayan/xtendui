Variant

Use tailwind classes to assign variant (e.g. border-color, background-color, text-color).

Size

Use tailwind classes to assign size (e.g. padding, font-size).

Checks Custom

You can have custom design that contains the input[type="checkbox"] or input[type="radio"], just add the tag label and then you can use tailwind variants on:, group-on: to assign animations.

For usability purpose hide the input inside with sr-only, don't assign display: none.

For usability purpose you can to apply focus-within styles to label, or wait for focus-visible-within.

Use button with label to create checks with button.

Use card with label to create checks with card.
