Wrap

You can wrap the slides with drag: { wrap: true }.

With drag: { wrap: true } you have to set loop: true and you can't use contain: true

You can set drag: { wrap: Number } where Number is how many times to cover the drag.dragger with wrap (setting 1 is the same as true).

It's useful sometimes if the wrap isn't long enough when dragging.

With drag: { wrap: true, overflow: false } you don't have the drag overflow on wrap.

Friction

You can set friction with drag: { friction: Function }, for example function(velocity) {return Math.pow(velocity, 0.9)}.

You can disable friction with drag: { friction: false }.

Overflow

You can set overflow with drag: { overflow: Function }, for example function(overflow) {return Math.pow(overflow, 0.73)}.

You can disable the drag transform position while dragging the drag.dragger with drag: { drag: false }.

False

Use drag: false to disable drag. You need to use the slider in toggle mode, so with .toggle-block on the slides.

.demo--slider--toggle-css { .slides-inner { overflow: hidden; // dragger animations .transition-none() !important; } .slide { .toggle-block(); } // animation .slide { &:after { content: ''; position: absolute; z-index: 9; top: 0; left: 0; bottom: 0; right: 0; background: @white; transform: translateX(0); } &.active { .anim-x-inverse.active(); animation-duration: @time-medium; &:after { animation-name: demo--slider--toggle-css--in; animation-duration: @time-medium; animation-timing-function: @bez-in; transform: translateX(-100%); } &.inverse { .anim-x.active(); animation-duration: @time-medium; &:after { animation-name: demo--slider--toggle-css-in-inverse; animation-duration: @time-medium; animation-timing-function: @bez-in; transform: translateX(100%); } } } &.out { .anim-x-inverse.out(); animation-duration: @time-medium; &:after { animation-name: demo--slider--toggle-css--out; animation-duration: @time-medium; animation-timing-function: @bez-out; transform: translateX(0); } &.inverse { .anim-x.out(); animation-duration: @time-medium; &:after { animation-name: demo--slider--toggle-css-out-inverse; animation-duration: @time-medium; animation-timing-function: @bez-out; transform: translateX(0); } } } } } @keyframes demo--slider--toggle-css--in { 0% { transform: translateX(0); } 100% { transform: translateX(-100%); } } @keyframes demo--slider--toggle-css-in-inverse { 0% { transform: translateX(0); } 100% { transform: translateX(100%); } } @keyframes demo--slider--toggle-css--out { 0% { transform: translateX(100%); } 100% { transform: translateX(0); } } @keyframes demo--slider--toggle-css-out-inverse { 0% { transform: translateX(-100%); } 100% { transform: translateX(0); } }

Custom drag animation with drag: false and javascript events.

.demo--slider--toggle-js { .slides-inner { overflow: hidden; // dragger animations .transition-none() !important; } .slide { .toggle-block(); } }

Xt.vars.timeMedium, translateX: xMax * direction, opacity: 0, ease: Xt.vars.easeOut }) gsap.to(self.dragger, { duration: Xt.vars.timeMedium, translateX: -xMax * direction, ease: Xt.vars.easeOut }) //
