Use overlay when you need to display any content over the main page, comes in different options and factors.

Screen

To make an overlay full screen use .xt-overlay-container.p-0 and .xt-card.h-screen and use tailwind classes (e.g. align-items, justify-content) to position .xt-card content.

Position

To make an overlay position use .xt-overlay-container.p-0 and .xt-card.h-screenand use tailwind classes (e.g. max-width, margin) on .xt-overlay-container.
