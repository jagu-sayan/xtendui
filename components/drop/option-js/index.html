Drop
Content

Drop uses toggle for logic, refer to toggle's option.

Class

You can start with toggled elements and targets just by adding one of the classes to the elements or targets.

It's recommended to use active-drop on drops otherwise the overlay flashes visible on page load.

SyntaxDefaultDescription
Optionclass:Stringactive active-dropClass name for activation

Event

You can specify on and off events for the drop. Fore example on: 'mouseenter' off: 'mouseleave'. Default is on: 'click'.

SyntaxDefaultDescription
Optionon:String'click'Set event to listen for activation
Optionoff:StringfalseSet event to listen for deactivation if not present options.on deactivate when already activated
OptioneventLimit:Query'.event-limit, .drop'Limit events inside self.object
