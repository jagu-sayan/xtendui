Core

Variant

Use tailwind classes to assign variant (e.g.: border, background, color).

Size

Use tailwind classes to assign text size.

Use component classes to assign size (e.g.: padding, font size).

SyntaxMixinDescription
Component.card-block-smcard-block-smSize small
Component.card-block-mdcard-block-mdSize medium
Component.card-block-lgcard-block-lgSize large

Use card classes to assign card size (e.g.: min-width).

SyntaxMixinDescription
Component.card-smcard-smWidth small
Component.card-mdcard-mdWidth medium
Component.card-lgcard-lgWidth large

Asset

If you want full width assets use .card-asset.

SyntaxMixinDescription
Component.card-assetcard-assetFull width assets

Block

Sequential .card-block stack vertically. Use tailwind classes to assign styles (e.g.: border-radius, background, spacing)

SyntaxMixinVariants
Component.card-blockcard-blockCard stack

Group

To stack .card-block horizontally wrap them inside .card-group.

SyntaxMixinVariantsDescription
Util.card-groupcard-groupresponsiveCard stack horizontal wrapper

The horizontal size of .card-block is automatic, but you can specify it with tailwind width.

Separator

Use tailwind classes to style borders for separation.

Overflow

Use tailwind classes to overflow divs. Use component classes to style overflow.

SyntaxMixinDescription
Component.overflow-mainoverflow-mainOverflow scrollbar style
Component.overflow-suboverflow-subOverflow scrollbar style small version

List

You can use list component to space card's internal content.
