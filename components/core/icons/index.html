Icons
Core

Everything you need to stylize svg icons.

Customization

You can customize this component inside tailwind.config.js, check xtendui/src/core/icons.css.js for default values.

module.exports = {
  theme:{
    xtendui: {
      icons: {
        component: theme => ({
          // modify components
        }),
      },
    },
  },
}
