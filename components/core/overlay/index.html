Overlay
Core

Used when you need to display any content over the main page.

Setup

Javascript

Follow the javascript installation instructions and import the component javascript file.

import 'xtend-ui/src/core/overlay'

Css

Follow the css installation instructions. To customize styles in your tailwind.config.js file.

module.exports = {
  theme: require('xtend-ui/src/tailwind-theme')({
    // xtend utilities and components in node_modules/xtend-ui/src/tailwind-xtend.js
    xtend: theme => ({
      utilities: {
        core: {
          overlay: {
            // modify utility
          },
        },
      },
      components: {
        core: {
          overlay: {
            // modify component
          },
        },
      },
    }),
  })
}

To see the default values see the source code of node_modules/xtend-ui/src/tailwind-xtend.js.

Overlays are moved to body to prevent z-index problems. Style and query overlay's content accordingly.

Usage

Pass elements and targets options if different from defaults. Elements listens to events, targets and inners activate on events.
