Disable

You can disable styles and javascript using .overlay-disable.

SyntaxMixinVariantsDescription
Util.overlay-disableoverlay-disable !importantresponsiveReset overlay styles and javascript
#demo--overlay-nested-disable-toggle { /*@apply md:hidden !important;*/ @apply hidden !important; } #demo--overlay-nested-disable { /*@apply md:overlay-disable !important;*/ @apply overlay-disable !important; }
import { Xt } from 'xtend-ui' Xt.mount.push({ matches: '#demo--overlay-nested-disable-toggle', mount: object => { // event const click = () => { document.querySelector('#demo--overlay-nested-disable').dispatchEvent(new CustomEvent('on.trigger.xt')) } object.addEventListener('click', click) }, })

Preset

You can overlay that covers the screen with .overlay-screen, remember to use also align and justify the .card.

SyntaxMixinVariantsDescription
Util.overlay-screenoverlay-screenresponsiveOverlay that covers the screen

With .overlay-screen you can also use tailwind classes to modify .overlay-container position and size.

Close fixed

You can have the .btn-close with fixed z-top to have it always visible also when scrolling.
