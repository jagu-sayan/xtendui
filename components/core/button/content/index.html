Content Button
Core

Simple and extensible button styles.

Variant

Use tailwind classes to assign variant (e.g. border-color, background-color, text-color).

Use .link to create a link.

Use .text-reset to create a button with inherited text.

Use tailwind classes to assign styles (e.g. text-color, text-opacity).

Text styles on parent elements are applied when inerithed.

See link variant to style links.

Size

Use component classes to assign size. You can add additional size for example .btn-xs or .btn-xl.

SyntaxMixinDescription
Component.btn-smbtn-smSize small
Component.btn-mdbtn-mdSize medium
Component.btn-lgbtn-lgSize large

Otherwise use tailwind classes to assign custom size (e.g. padding, font-size).

Icons and Alignment

Use icons as content.

Use tailwind classes to assign alignment (e.g. align-items, justify-content, text-align).
