List
Core

Flex component to space any element with margins, also spaces vertically.

Customization

You can customize this component inside tailwind.config.js, check xtendui/src/core/list.css.js for default values.

module.exports = {
  theme:{
    xtendui: {
      list: {
        utility: theme => ({
          // modify utilities
        }),
        component: theme => ({
          // modify components
        }),
      },
    },
  },
}
