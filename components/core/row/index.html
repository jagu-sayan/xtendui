Row
Core

Flex component to space columns with paddings, also spaces vertically.

Customization

You can customize this component inside tailwind.config.js, check xtendui/src/core/row.css.js for default values.

module.exports = {
  theme:{
    xtendui: {
      row: {
        utility: theme => ({
          // modify utilities
        }),
        component: theme => ({
          // modify components
        }),
      },
    },
  },
}

Usage

Use this code to create a row.

You should always use tailwind width classes inside .row. Otherwise you can space contents with list.
