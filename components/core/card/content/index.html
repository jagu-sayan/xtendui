Content Card
Core

Multi purpose component to contain content.

Variant

Use tailwind classes to assign variant (e.g. border-color, background-color, text-color).

Interactive

Use tailwind classes to modify variants for interactive animations.

Size

Use component classes to assign size. You can add additional size for example .card-xs or .card-xl.

SyntaxMixinDescription
Component.card-smcard-smSize small
Component.card-mdcard-mdSize medium
Component.card-lgcard-lgSize large

Otherwise use tailwind classes to assign custom size (e.g. padding, font-size).

Content

Card content stack vertically. Use tailwind classes to assign styles (e.g. padding, background-color, border-radius)

Group

Use component utility .card-group to stack card content horizontally.

SyntaxMixinVariantsDescription
Utility.card-groupcard-groupresponsiveCard stack horizontal wrapper

Use tailwind classes to assign size (e.g. width).

Separator

Use tailwind classes to style borders for separation.

Overflow

Use tailwind classes to overflow divs. Use component classes to style overflow.

SyntaxMixinDescription
Component.overflow-mainoverflow-mainOverflow scrollbar style
Component.overflow-suboverflow-subOverflow scrollbar style small version

List

You can use list component to space card's internal content.

Close button

Use tailwind classes to assign size (e.g. padding, font-size).
