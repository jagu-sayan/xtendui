Content Table
Core

Base styles for all tables overridable with tailwind classes.

Variant

Use tailwind classes to assign variant (e.g. border-color, background-color, text-color).

Interactive

Use tailwind classes to assign animation (e.g. translate, transition-property, transition-duration).

Size

Use tailwind classes to assign custom size (e.g. padding, font-size).

Overflow

Use component classes to overflow with scroll when content is too long (use narrow screen to see it in action).
