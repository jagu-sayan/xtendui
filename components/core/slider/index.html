Slider
Core

Create visually stunning sliders, has a comprehensive javascript api.

Installation

Import the component javascript file.

import 'xtendui/src/core/slider'

Customization

You can customize this component inside tailwind.config.js, check xtendui/src/core/slider.css.js for default values.

module.exports = {
  theme:{
    xtendui: {
      slider: {
        utility: theme => ({
          // modify utilities
        }),
        component: theme => ({
          // modify components
        }),
      },
    },
  },
}

Usage

Use this code to create a slider.

[data-xt-pag].xt-ignore is essential to the functioning of the slider, but you can hide it using the class .hidden.

Check subpage to customize interaction and customize position.
